# Docsify默认页面

> 启动后台运行命令
>
> `docsify serve docs &`

# 插入图片
> 插入图片方法
>
> `<img src="图片文件目录>`
>
> `<img src="img/logo.jpg">`
# 插入代码
> 插入代码方法
>
> 1. 插入单行代码头尾插入一个[`]
> 2. 插入多行代码头尾插入一个[```]
>
>
# 插入链接
> 连接名称
>
> `[连接名称](http://www.baidu.com)`
>
> `[连接名称](连接地址)`

# 插入表格
> 插入表格方法
>
>```linux
>| 标题 | 名称 | 类容 |

>| 内容 | 内容 | 内容 |
>| 内容 | 内容 | 内容 |
>```

## Node安装
> ```linux
> sudo npm install n -g 
> sudo n stable 升级到最新版本
> ```
> **n是一个Node工具包，它提供了几个升级命令参数**
> 1. n 显示已安装的Node版本
> 2. n latest 安装最新版本Node
> 3. n stable 安装最新稳定版Node
> 4. n lts 安装最新长期维护版(lts)Node
> 5. n version 根据提供的版本号安装Node
